1. On your command prompt, type command:
    ssh-keygen
2. Then change the directory to .ssh using command:
    cd .ssh
3. Then view contents of id_rsa.pub file using command(command may differ for different operating systems):
    type id_rsa.pub 
4. Copy the contents/output
5. Login to your github account and go to settings.
6. Under settings, go to SSH and GPG keys and add new SSH key.
